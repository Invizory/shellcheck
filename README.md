# ShellCheck for CI

[ShellCheck] [Docker] image for CI that automatically finds scripts to check.

[ShellCheck]: https://www.shellcheck.net
[Docker]: https://www.docker.com

## Usage

    docker run -tv "$(pwd):/source:ro" inviz/shellcheck
